# React GIF App

Aplicación web para buscar y mostrar GIFs utilizando **React, TypeScript y Vite**. La app consume la API de Giphy (https://developers.giphy.com/explorer/) para obtener GIFs a partir de un término de búsqueda ingresado por el usuario y mostrarlos dinámicamente en la interfaz. El proyecto utiliza una **estructura modular**, separando componentes y lógica para mantener el código organizado y escalable. La obtención de datos se maneja mediante **custom hooks**, lo que permite encapsular la lógica de las peticiones y reutilizarla en diferentes partes de la aplicación siguiendo buenas prácticas de desarrollo en React. También se implementa un **sistema de cache en memoria** para evitar solicitudes repetidas a la API, mejorar el rendimiento y permitir que los GIFs ya consultados se recuperen sin necesidad de recargar la página. El código sigue **principios de código limpio**, separación de responsabilidades y reutilización de lógica para facilitar el mantenimiento y la escalabilidad del proyecto. Para instalar las dependencias ejecuta `npm install`. Para iniciar el servidor de desarrollo usa `npm run dev` y abre `http://localhost:5173` en el navegador. Para generar la versión de producción ejecuta `npm run build` y para previsualizar el build usa `npm run preview`.
